# AuraSpend 💸

A world-class expense manager designed for Android 16 with a high-end UI/UX, seamless performance, and cloud synchronization.

## ✨ Features
- **Pure AMOLED Mode**: True black theme for OLED screens (Premium).
- **Edge-to-Edge UI**: Modern, immersive experience.
- **Google Drive Sync**: Secure cloud backups and synchronization.
- **MVI Architecture**: Predictable state and fluid animations.
- **Advanced Analytics**: Deep insights into spending habits (Premium).
- **Lottie Animations**: Beautiful onboarding and splash experience.

## 🛠 Tech Stack
- **Language**: Pure Kotlin
- **UI**: Jetpack Compose
- **Architecture**: Clean Architecture + MVI
- **Local Storage**: Room Database
- **Cloud**: Google Drive API
- **Target SDK**: Android 16 (API 36)

## 🚀 Getting Started
### Prerequisites
- Android Studio Koala or newer
- JDK 17+
- Android SDK 36 (Android 16)

### Setup
1. Clone the repository.
2. Open in Android Studio.
3. Sync Gradle.
